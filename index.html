libmormegil by fluffymormegil

Common library code intended for use across multiple fluffymormegil projects.

libmormegil is a collection of C++ library and header files (including C interfaces in some cases)

Dependencies

A working C++ environment which includes a C99 stdint.h header that includes uint64_t

Install

./configure --prefix /my/install/location && make all && make install

Of course, if I can get you to do that just by telling you to on my web page, your system has a security hole :)
